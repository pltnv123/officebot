# 3D Models Resources for OfficeBot

## 🎨 Quality Standards (Pixar-like)

### Requirements:
- Game-ready (low poly, <10K triangles)
- Clean topology
- PBR materials (or stylized shader compatible)
- FBX/OBJ format (Unity compatible)

---

## 🆓 Free Sources (CC0 / Free License)

### 1. Kenney.nl (CC0)
- **URL:** https://www.kenney.nl/assets
- **Quality:** ★★★★☆
- **Style:** Low-poly, game-ready
- **Best for:** Environment, props, UI elements
- **Download:** Direct, no registration

### 2. Poly Haven (CC0)
- **URL:** https://polyhaven.com/models
- **Quality:** ★★★★★
- **Style:** Photorealistic
- **Best for:** HDRIs, textures, environments
- **Download:** Direct, no registration

### 3. Sketchfab (Mixed)
- **URL:** https://sketchfab.com/search?type=models&q=robot
- **Quality:** ★★★★★
- **Style:** Various (filter by CC/Buy)
- **Best for:** Character models, detailed robots
- **Download:** Requires account for some

### 4. Free3D
- **URL:** https://free3d.com/3d-models/robot
- **Quality:** ★★★☆☆
- **Style:** Various
- **Best for:** Quick prototypes
- **Download:** Free registration

### 5. Mixamo (Adobe)
- **URL:** https://www.mixamo.com/
- **Quality:** ★★★★★
- **Style:** Character animations
- **Best for:** Robot animations (idle, walking)
- **Download:** Free with Adobe account
