# Agency Agents Activation Stages

Each imported role progresses through defined stages before it can drive runtime behavior.

- **Stage 0 (Imported, Dormant):** Cataloged but not exposed; serves docs reference and stays dormant.
- **Stage 1 (Docs-visible):** Listed in operator materials and routing maps, but still non-selectable.
- **Stage 2 (Selectable):** Positions become eligible for manual binding/testing without auto-wiring.
- **Stage 3 (Runtime-wired):** Roles are wired via bridge/config and become live in specific flows.

Transitions happen manually to preserve stability and prevent mass activation.
