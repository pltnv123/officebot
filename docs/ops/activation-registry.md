- technical-director → `third_party/ccgs/agents/technical-director.md`
- release-manager → `third_party/ccgs/agents/release-manager.md`
- qa-lead → `third_party/ccgs/agents/qa-lead.md`
- devops-engineer → `third_party/ccgs/agents/devops-engineer.md`
- lead-programmer → `third_party/ccgs/agents/lead-programmer.md`
