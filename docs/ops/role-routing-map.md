# Role Routing Map

- technical-director -> architecture / prioritization / technical tradeoffs
- release-manager -> release gates / rollout readiness / deployment coordination
- qa-lead -> verification / test planning / evidence expectations
- devops-engineer -> infra / automation / pipeline reliability
- lead-programmer -> implementation planning / code-shaping / integration guidance

## Rule
All imported agents beyond these five remain vendor/reference only until later activation.
