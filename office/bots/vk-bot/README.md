# 🤖 pltnv AI — VK Bot

VK бот для сообщества AI-продуктов. Постит контент, отвечает на комментарии, приветствует новых участников.

## 📁 Структура

```
vk-bot/
├── bot.py              # Основной файл бота
├── content_planner.py   # План контента (3-5 постов/день)
├── community_manager.py # Модерация и ответы
├── human.py            # Human-like модуль (сленг, эмодзи)
├── config.py           # Настройки
└── README.md           # Эта документация
```

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install vk_api
```

### 2. Настройка

Откройте `config.py` и заполните:

```python
VK_TOKEN = "ваш_токен_сообщества"
GROUP_ID = 123456789  # ID вашего сообщества
```

**Как получить токен:**
1. Зайдите в настройки сообщества
2. Работа с API → Ключи доступа сообщества
3. Создайте ключ с правами: wall, messages, manage

### 3. Запуск

```bash
python bot.py
```

## 📋 Функционал

### Посты по расписанию

| Время | Тип контента |
|-------|-------------|
| 09:00 | Доброе утро + совет |
| 13:00 | Кейс/история |
| 19:00 | Интерактив/опрос |
| 21:00 | Итоги/полезное |

### Типы постов

- 📝 **Текстовые** — советы, кейсы, лайфхаки
- 📊 **Опросы** — высокая вовлечённость в VK
- 🖼️ **Картинки** (требует расширения)
- 📹 **Видео** (требует расширения)

### Автоответы

Бот отвечает на:
- Личные сообщения
- Комментарии к постам
- Вопросы из FAQ (цены, услуги, контакты)

### Модерация

Автоматическое:
- Блокировка спама
- Удаление запрещёнки
- Игнор бессмысленных комментариев

### Приветствия

Новые участники получают приветственное сообщение в ЛС.

## 🎨 Human-like стиль

Бот пишет как живой человек:
- Casual тон: "привет!" не "здравствуйте"
- Умеренные эмодзи
- Русский сленг: "спс", "ок", "го"
- Паузы между сообщениями (имитация набора)

## ⚙️ Настройка

Основные параметры в `config.py`:

```python
# Расписание
SCHEDULE = {
    "morning": "09:00",
    "noon": "13:00",
    "evening": "19:00",
    "night": "21:00",
}

# Задержки
MESSAGE_DELAY_MIN = 1.5
MESSAGE_DELAY_MAX = 4.0

# Модерация
BANNED_WORDS = ["слово1", "слово2"]
```

## 🔧 Расширение

### Добавить новые посты

Редактируйте `content_planner.py` — списки `MORNING_POSTS`, `NOON_POSTS`, и т.д.

### Добавить FAQ

В `community_manager.py` — список `FAQ_RESPONSES`:

```python
{
    "patterns": [r"ваш.*паттерн"],
    "reply": "ваш ответ",
},
```

### Добавить приветствия

В `community_manager.py` — список `WELCOME_MESSAGES`.
