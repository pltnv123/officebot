# Бизнес-план: OfficeBot — AI-агенты с визуальной сценой

## Продукт
Персональный AI-ассистент на отдельном VPS с 3D-визуализацией работы агентов.
Клиент получает: свой сервер, свой бот, свою 3D-сцену, своих агентов.

## Что видит пользователь
- Telegram-бот, который реально помогает (задачи, напоминания, развитие, контент)
- Мобильное приложение с 3D-офисом, где видно работу агентов в реальном времени
- Ощущение: «у меня есть своя AI-команда»

## Техническая архитектура (per client)
```
VPS (4GB RAM) → $5-10/мес
├── OpenClaw (multi-agent orchestrator)
├── Tor + Privoxy (анонимность, обход блокировок)
├── Unity WebGL сцена (порт 80/443)
├── Telegram Bot (персональный)
├── Backend API (состояние агентов)
└── Агенты: Planner, Worker, Reviewer, Builder
```

## Что нужно сделать для запуска

### Техническое (2-3 недели)
1. Автоматизированное развёртывание VPS (Ansible/Terraform)
2. Скрипт установки OpenClaw + настройка агентов
3. Мобильное приложение (обёртка WebView для WebGL сцены)
4. Личный кабинет клиента (оплата, настройка)
5. Документация и онбординг

### Маркетинговое
1. Лендинг с демо-видео
2. Презентация для потенциальных клиентов
3. Канал продаж: ProductHunt, Twitter/X, Telegram-каналы
