## 1. Описание продукта

OfficeBot — это **личный AI-помощник** в формате красивого 3D-приложения (Unity), доступного в браузере, на телефоне и в Telegram. Не аналог ChatGPT и не корпоративный инструмент. Это персональный спутник, который помогает развиваться, решать сложные вопросы и становиться лучше.

**Ключевая метафора:** у тебя есть свой офис, свои роботы-помощники, свои комнаты. Это не чат-бот — это **виртуальная среда** с живыми агентами.

## 3. Модель ценообразования

**Подписка**, понятная для русскоязычного пользователя.

Принципы:
- **Не дорогая** — чтобы пользователи возвращались, а не уходили после месяца
- Примерный диапазон: **299–999 ₽/месяц** (или $5–$10)
- Первый месяц / пробный период — обязательно
- Крипто-платёж для ранних адептов (сейчас $100 в крипте как базовый порог)

## 5. Ключевые отличия (UVP)

### 5.1 Визуальная среда на Unity
- Красивое 3D-приложение, а не веб-форма
- Мобильное приложение (iOS / Android)
- WebGL-версия для браузера
- Постоянно под рукой: браузер, телефон, Telegram

### 5.2 AI-роботы-агенты
- Каждый робот — отдельный агент с ролью (WORKER, PLANNER, REVIEWER...)
- Взаимодействуют между собой как механизм
- Модели роботов генерируются AI (3D-генерация)
- Каждый робот можно настраивать

### 5.3 Система комнат
- Комнаты = отделы / пространства / контексты
- Комната скриптов и программ
- Пользователь добавляет свои комнаты
- Комната определяет контекст работы агентов

### 5.4 Социальные функции
- Друзья в приложении
- Score / рейтинг
- Бонусы за активность
- Совместная работа в комнатах

### 5.5 Полезные скрипты и программы «в коробке»
- Готовые автоматизации
- Пользовательские скрипты
- Маркетплейс инструментов

## 6. Техническая архитектура (высокоуровнево)

```
┌─────────────────────────────────────────────┐
│              Клиенты                        │
│  ┌──────┐  ┌──────────┐  ┌──────────────┐  │
│  │ WebGL│  │ Мобильное│  │   Telegram   │  │
│  │Unity │  │  прилож. │  │    бот/mini  │  │
│  └──┬───┘  └────┬─────┘  └──────┬───────┘  │
└─────┼───────────┼────────────────┼──────────┘
      │           │                │
┌─────▼───────────▼────────────────▼──────────┐
│              API Gateway                     │
│         (REST + WebSocket)                  │
└─────┬───────────┬────────────────┬──────────┘
      │           │                │
┌─────▼───┐  ┌────▼─────┐  ┌──────▼───────┐
│ Agent   │  │  State   │  │    Social    │
│ Engine  │  │  Store   │  │   Service    │
│(Multi-  │  │ (per-user│  │  (friends,   │
│ agent)  │  │  server) │  │   score)     │
└─────────┘  └──────────┘  └──────────────┘
      │           │
┌─────▼───────────▼──────────────────────────┐
│         Per-User Server (Docker)            │
│   Каждый пользователь получает свой контейнер│
│   Оплачивается через подписку               │
└─────────────────────────────────────────────┘
```

### Принципы архитектуры:
1. **Каждому пользователю — свой сервер** (Docker-контейнер)
2. Сервер оплачивается через подписку
3. Агенты работают внутри контейнера пользователя
4. AI-генерация 3D-моделей через внешние API (Meshy.ai, Tripo3D и др.)
