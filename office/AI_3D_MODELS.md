# AI-генерация 3D-моделей для OfficeBot

> Дата: 2026-03-17
> Назначение: Исследование инструментов генерации 3D-моделей роботов для Unity

---

## Обзор

Для OfficeBot нужны 3D-модели роботов-агентов. Антон определил: **AI-генерация моделей — лучший вариант**. Это позволяет:
- Быстро создавать уникальных роботов
- Масштабировать без 3D-художников
- Давать пользователям возможность кастомизировать роботов
- Поддерживать разнообразие в экосистеме

---

## 1. Meshy.ai

**Сайт:** https://www.meshy.ai

### Возможности
- Text-to-3D и Image-to-3D генерация
- AI-текстурирование
- Автоматическая ретопология
- Экспорт в GLB, FBX, OBJ, USDZ, STL
- API для интеграции
- Unity-плагин (прямая интеграция)

### Тарифы (март 2026)

| План | Цена | Кредиты/мес | Скачиваний | Очередь |
|------|------|-------------|------------|---------|
| **Free** | $0 | 100 | 10 (Meshy-4) | 1 задача, низкий приоритет |
| **Pro** | ~$20/мес | 1,000 (до 100 ассетов) | Безлимит | 10 задач, высокий приоритет |
| **Studio** | ~$60/мес | 4,000 (до 400 ассетов) | Безлимит | 20 задач, командный доступ |
| **Enterprise** | Кастомный | Кастомный | Безлимит | 50+ задач |

### Качество
- **Meshy-4** — последняя модель, лучшее качество
- Хорошо для стилизованных персонажей (cartoon, low-poly)
- Текстуры генерируются автоматически
- Для роботов — **отлично подходит** (геометрические формы, чёткие линии)

### Интеграция с Unity
- Официальный плагин
- Прямой импорт GLB/GLTF
- Автоматическая конвертация материалов

### Вердикт для OfficeBot: ⭐⭐⭐⭐⭐
> Лучший выбор. API + Unity-плагин + хорошее качество для роботов-стилизованных персонажей.

- AI-видеогенерация (Dream Machine / Ray3.14)
- Photogrammetry (3D-сканирование через видео)
- Вывод 3D-сцены из видео

### Тарифы (март 2026)

| План | Цена | Кредиты | Примечание |
|------|------|---------|------------|
| **Free** | $0 | Пробные кредиты | Ограниченный доступ |
| **Plus** | ~$24/мес | Базовые | Коммерческое использование |
| **Pro** | ~$59/мес | 4x кредитов | Приоритетная генерация |
| **Business** | Кастомный | Командный | SSO, аналитика |

Стоимость генерации:
- Видео (Ray3.14, 540p): 10 кредитов/сек
- Видео (1080p): 80 кредитов/сек

- Для стилизованных роботов — **средне** (не основной фокус)

### Интеграция с Unity
- Нет прямого плагина
- Экспорт GLB/OBJ вручную
- API доступен

### Вердикт для OfficeBot: ⭐⭐⭐
> Хорошая технология, но фокус на видео, а не 3D-модели. Можно использовать для маркетинговых видео с роботами, но не как основной генератор моделей.

**Сайт:** https://www.tripo3d.ai

### Возможности
- Text-to-3D и Image-to-3D
- **AI Rigging** (скелетная анимация) — ключевая фича!
- AI-текстурирование
- Multi-view генерация (несколько ракурсов)
- Batch-генерация
- Part segmentation и completion
- Smart Low-Poly (оптимизация полигонов)
- Стилизация: LEGO, voxel, Minecraft, clay, cartoon

### Тарифы (март 2026)

| План | Цена | Кредиты/мес | Concurrent | Хранение |
|------|------|-------------|------------|----------|
| **Basic** | $0 | 300 | 1 | 20 моделей |
| **Professional** | ~$12/мес | 3,000 | 10 | 60 моделей |
| **Advanced** | ~$30/мес | 8,000 | 15 | 200 моделей |
| **Premium** | ~$84/мес | 25,000 | 20 | Безлимит |

### Качество
- **Tripo v3.0 Ultra** — последняя модель
- Отлично для персонажей и существ
- **AI Rigging** — автоматическое добавление скелета для анимации
- Стилизация впечатляет (LEGO, clay, cartoon)
- Для роботов — **отлично**, особенно с анимацией

### Интеграция с Unity
- Экспорт в GLB, FBX (поддерживает скелет)

- Прямого Unity-плагина нет, но FBX импортируется natивно

> **Лучший выбор для анимированных роботов.** AI Rigging — ключевая фича: роботы могут анимироваться без ручного rigging. Стилизация хорошо подходит к концепции.

## Рекомендация для OfficeBot

### Primary: Tripo3D
**Причины:**
1. **AI Rigging** — роботы могут анимироваться сразу после генерации
2. Лучшее качество для персонажей-роботов
3. Доступная цена ($12/мес Professional)
4. Стилизация хорошо вписывается в концепцию
5. Batch-генерация для создания множества роботов

### Secondary: Meshy.ai
**Использование:**
- Unity-плагин для прямого импорта
- Резервный провайдер
- Генерация простых объектов для комнат

### API-стратегия

```python
# Примерная архитектура генерации роботов

class RobotGenerator:
    def __init__(self):
        self.primary = Tripo3DClient(api_key=...)
        self.backup = MeshyClient(api_key=...)
    
    async def generate_robot(self, description: str, style: str = "cartoon"):
        """Генерация 3D-модели робота по описанию"""
        
        # 1. Генерация основной модели через Tripo3D
        model = await self.primary.text_to_3d(
            prompt=f"{style} robot, {description}",
            model="v3.0_ultra",
            style="cartoon"  # или clay, lego, etc.
        )
        
        # 2. AI Rigging (только Tripo3D)
        rigged = await self.primary.auto_rig(
            model_id=model.id,
            rig_type="humanoid"  # или "quadruped" для четырёхногих
        )
        
        # 3. Текстурирование
        textured = await self.primary.apply_texture(
            model_id=rigged.id,
            style=style
        )
        
        # 4. Экспорт в FBX для Unity
        fbx_path = await self.primary.export(
            model_id=textured.id,
            format="fbx",
            include_skeleton=True
        )
        
        return fbx_path
```

### Unity Integration Pipeline

```
Tripo3D API → FBX + Textures → Unity Import Pipeline
                                    ↓
                            Auto-setup Materials
                                    ↓
                            Attach Animator
                                    ↓
                            Robot Prefab ready
```
