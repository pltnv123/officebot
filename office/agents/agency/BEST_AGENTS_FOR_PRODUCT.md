# Лучшие агенты для OfficeBot Product

## Критерии отбора:
- Полезность для обычного пользователя
- Уникальность (нет в ChatGPT/Copilot)
- Реализуемость в Unity WebGL

---

## 🏆 TOP 10 АГЕНТОВ (для MVP)

### 1. 🧑‍💻 Software Architect (eng-software-architect.md)
**Почему:** Проектирование систем, архитектурные решения
**Полезность:** Высокая — помогает проектировать проекты

### 2. 📝 Technical Writer (eng-technical-writer.md)
**Почему:** Документация, README, инструкции
**Полезность:** Высокая — все нужны документы

### 3. 🔒 Security Engineer (eng-security-engineer.md)
**Почему:** Проверка безопасности, рекомендации
**Полезность:** Высокая — безопасность важна

### 4. 🎨 UI Designer (design-ui-designer.md)
**Почему:** Дизайн интерфейсов, макеты
**Полезность:** Высокая — все делают интерфейсы

### 5. 🎮 Game Designer (gamedev-game-designer.md)
**Почему:** Геймдизайн, механики
**Полезность:** Средняя — для нашей сцены

### 6. 📊 Product Manager (prod-manager.md)
**Почему:** Roadmap, приоритизация, стратегия
**Полезность:** Высокая — для планирования

### 7. 🧪 API Tester (test-api-tester.md)
**Почему:** Тестирование API, автоматизация
**Полезность:** Высокая — QA важен

### 8. 🚀 Growth Hacker (mkt-growth-hacker.md)
**Почему:** Рост, маркетинг, эксперименты
**Полезность:** Средняя — для продвижения

### 9. 💼 Deal Strategist (sales-deal-strategist.md)
**Почему:** Продажи, переговоры
**Полезность:** Средняя — для бизнеса

### 10. 🎯 Content Creator (mkt-content-creator.md)
**Почему:** Контент, посты, статьи
**Полезность:** Высокая — контентмаркетинг
