# OfficeBot Twitter Agent 🤖🐦

Human-like Twitter бот для продвижения OfficeBot. Пишет посты, отвечает на комментарии, лайкает и ретвитит релевантный контент — всё как настоящий человек.

## Особенности

- 🧠 **Human-like поведение** — случайные паузы, опечатки (1%), сленг, эмодзи
- 📝 **AI-генерация контента** — OpenRouter API (Claude)
- 🎯 **5 тематик** — AI, продуктивность, разработка, бизнес, личный опыт
- 📊 **Лимиты** — защита от спама и банов
- ⏰ **Временные окна** — постит утром и вечером как человек
- 🔄 **Engagement** — автоматические лайки, ретвиты, ответы на упоминания

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install tweepy requests
```

### 2. Настройка

Отредактируй `config.py`:

```python
# Twitter API Credentials (получить на developer.twitter.com)
TWITTER_CONFIG = {
    "api_key": "твой_api_key",
    "api_secret": "твой_api_secret",
    "access_token": "твой_access_token",
    "access_token_secret": "твой_access_token_secret",
    "bearer_token": "твой_bearer_token",
}

# OpenRouter API (получить на openrouter.ai)
OPENROUTER_CONFIG = {
    "api_key": "твой_openrouter_key",
}
```

### 3. Запуск

```bash
# Тестовый режим (без реальных твитов)
python bot.py --test

# Один цикл
python bot.py --once

# Основной цикл
python bot.py

# Статистика
python bot.py --stats

# Опубликовать конкретный текст
python bot.py --post "мой твит"
```

## Как это работает

### Генерация контента

Бот генерирует посты через OpenRouter API с промптом Persona:

```
Ты — Макс, 28, фрилансер-разработчик.
Пишешь неформально, как обычный человек.
Делишься конкретным опытом.
```

### Human-like особенности

| Фича | Описание |
|------|----------|
| Паузы | 1-4 часа между постами (случайно) |
| Опечатки | 1% шанс (klava shift) |
| Время | Постит 7-10 и 18-22 |
| Выходные | Реже постит (30% активности) |
| Сленг | "короче", "блин", "типа того" |
| Формат | Без кавычек, строчные буквы, эмодзи |

### Дневные лимиты

- Посты: 8
- Лайки: 50
- Ретвиты: 15
- Ответы: 20

## Безопасность

⚠️ **Важно:**
- Никогда не коммить credentials
- Используй environment variables для продакшена
- Twitter банит за спам — бот ограничен лимитами
- Human-like паузы защищают от детекции

### Environment variables (альтернатива)

```bash
export TWITTER_API_KEY="..."
export TWITTER_API_SECRET="..."
export TWITTER_ACCESS_TOKEN="..."
export TWITTER_ACCESS_TOKEN_SECRET="..."
export TWITTER_BEARER_TOKEN="..."
export OPENROUTER_API_KEY="..."
```
