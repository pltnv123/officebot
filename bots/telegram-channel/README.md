# OfficeBot Channel Manager

Telegram бот для ведения канала про AI и автоматизацию.

## Что делает

- 📝 Публикует посты по расписанию (5 раз в день)
- 💬 Отвечает на комментарии под постами
- 📊 Создаёт опросы и викторины
- 🤖 Ведёт себя как живой человек

## Расписание постов (МСК)

| Время | Тип |
|-------|-----|
| 09:00 | Доброе утро + совет по AI |
| 12:00 | Кейс/история успеха |
| 15:00 | Туториал/инструкция |
| 18:00 | Интерактив (опрос/викторина) |
| 21:00 | Итоги дня |

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

В `config.py`:
- `BOT_TOKEN` — токен бота от @BotFather
- `CHANNEL_ID` — ID канала (например `@officebot_ai`)
- `OPENROUTER_API_KEY` — ключ для AI генерации

## Запуск

```bash
python bot.py
```

## Команды (для админа в ЛС)

- `/post` — опубликовать пост вручную
- `/post case` — кейс
- `/post tutorial` — туториал
- `/post interactive` — интерактив
- `/poll` — создать опрос
- `/status` — статистика канала
- `/schedule` — показать расписание

## Особенности

- **Human-like стиль** — посты как от живого человека
- **AI генерация** — уникальный контент через OpenRouter
- **Автоматические ответы** — на комментарии отвечает бот
- **Планировщик** — посты по расписанию
- **Fallback модели** — если основная модель недоступна
